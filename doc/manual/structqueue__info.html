Function: clear

Usage: myqueue.clear(&myqueue);

This function clear all the data in the queue. So the queue object can later be reused. If you wanna dispose this queue, DON NOT use it, use destory() instead. (Or you'll get a leakage)

int(* queue_info::pop) (queue *this, void *target)

Function: pop

Usage: myqueue.pop(&myqueue, &buffer);

This function copy the first element into the datafield pointed by &buffer. And then DELETE the first element in the queue. In case of empty queue, it returns a non-zero value (POP_IN_EMPTY_QUEUE), if everything is OK, it shall return 0

◆ push

int(* queue_info::push) (queue *this, const void *data)

Function: push

Usage: myqueue.push(&myqueue, &input);

This function COPY the data pointed by a pointer into a new element in the queue. Later, when variable "input" changes its value, the element in queue will NOT change. if everything is OK, it shall return 0, otherwise, it returns a non-zero value indicating an error.
